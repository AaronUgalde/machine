# Ejercicio: Clasificador Lineal (Introducción a SVM)

Este notebook implementa un clasificador lineal simple para separar dos conjuntos de datos bidimensionales.

## Descripción

El programa realiza los siguientes pasos:

1.  **Definición de Datos:** Se definen dos clases de puntos: `positivos` y `negativos`.
2.  **Cálculo de Vectores Promedio:** Se calculan los vectores promedio para cada clase de puntos.
3.  **Visualización:** Se grafican los puntos y sus vectores promedio para visualizar su distribución.
4.  **Definición del Hiperplano:** Se determina una línea (hiperplano) que separa las dos clases.
5.  **Clasificación:** Se implementa una función para clasificar nuevos puntos, determinando si pertenecen a la clase "Positivo" o "Negativo" según su posición con respecto a la línea de separación.

Este ejercicio sirve como una introducción conceptual a los principios de las Máquinas de Vectores de Soporte (SVM) al encontrar un hiperplano que divide un espacio de características.
